Método getShort (java.lang.String)

Recupera el valor del parámetro designado como un valor short en el lenguaje de programación Java según el nombre del parámetro.

Sintaxis

public short getShort(java.lang.String sCol)

Parámetros

sCol

Un String que contiene el nombre del parámetro.

Valor devuelto

Un valor short .

Excepciones

SQLServerException

Comentarios

El método getShort especifica este método getShort en la interfaz java.sql.CallableStatement.

Este método solamente se admite en los tipos de datos SQL Server que pueden devolver de forma segura un valor entero como smallint, tinyint y bit. Si se utiliza este método en cualquier otro tipo de datos, provocará una excepción.
