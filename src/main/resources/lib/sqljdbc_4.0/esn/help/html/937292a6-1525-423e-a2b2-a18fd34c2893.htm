Clase SQLServerConnection

Representa una conexión JDBC a una base de datos de MicrosoftSQL Server.

Paquete: com.microsoft.sqlserver.jdbc

Implementa:ISQLServerConnection, java.io.Serializable

Sintaxis

public class SQLServerConnection

Comentarios

SQLServerConnection admite la agrupación de conexiones JDBC y se puede tratar de una conexión JDBC física o de una conexión JDBC lógica. SQLServerConnection administra el control de transacciones para todas las instrucciones que se crearon a partir de ella misma y puede participar en transacciones XA distribuidas que se administren a través de un adaptador XAResource.

SQLServerConnection administra un grupo de identificadores de instrucciones preparados. Las instrucciones preparadas se generan una vez y, por lo general, se ejecutan muchas veces con valores de datos diferentes para sus parámetros. Las instrucciones preparadas también se mantienen en cierres de conexiones lógicas (agrupadas).

SQLServerConnection no es un subproceso seguro. Sin embargo, se pueden procesar varias instrucciones que se crean a partir de una única conexión de forma simultánea en subprocesos que se desarrollan a la vez.

Esta clase admite la acción de desencapsular para la clase SQLServerConnection, la interfaz java.sql.connection y la interfaz ISQLServerConnection. Para obtener más información, vea Contenedores e interfaces.
