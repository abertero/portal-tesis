Obtener la versión del controlador

La versión del Controlador Microsoft JDBC para SQL Server instalado se puede encontrar de las maneras siguientes:

  • Llame a los métodos SQLServerDatabaseMetaData, getDriverMajorVersion, getDriverMinorVersion o getDriverVersion.

  • La versión se muestra en el archivo readme.txt de la distribución del producto.

Además, el nombre del controlador JDBC se puede devolver desde la llamada al método getDriverName de la clase SQLServerDatabaseMetaData. Devolverá "Controlador Microsoft JDBC 4.0 para SQL Server".

A continuación se muestra un ejemplo del resultado de las llamadas a los métodos de la clase SQLServerDatabaseMetaData:

getDriverName = Microsoft JDBC Driver 4.0 for SQL Server

getDriverMajorVersion = 4

getDriverMinorVersion = 0

getDriverVersion = 4.0.xxx.x

Donde "xxx.x" es el número de la versión final.
