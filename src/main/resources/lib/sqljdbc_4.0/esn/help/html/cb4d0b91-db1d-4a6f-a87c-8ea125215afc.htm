Método supportsMultipleResultSets (SQLServerDatabaseMetaData)

Recupera si esta base de datos admite recibir varios objetos SQLServerResultSet a partir de una llamada única al método execute de la clase SQLServerCallableStatement.

Sintaxis

public boolean supportsMultipleResultSets()

Valor devuelto

true si se admite. En caso contrario, es false.

Excepciones

SQLServerException

Comentarios

El método supportsMultipleResultSets especifica este método supportsMultipleResultSets en la interfaz java.sql.DatabaseMetaData.
