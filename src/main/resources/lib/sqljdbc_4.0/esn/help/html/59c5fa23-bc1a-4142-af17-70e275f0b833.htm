Método setPortNumber (SQLServerDataSource)

Establece el número de puerto que se va a utilizar para la comunicación con SQL Server.

Sintaxis

public void setPortNumber(int portNumber)

Parámetros

portNumber

Un valor int que contiene el número de puerto.

Comentarios

El número de puerto es el número del puerto TCP/IP que se utiliza al abrir una conexión de socket para SQL Server. Si no se establece la propiedad portNumber, el método getPortNumber devuelve el valor predeterminado de 1433.

El método setPortNumber no efectúa ninguna comprobación del intervalo en el valor del puerto que se ha pasado. Puede pasar un número de puerto que no es válido, como 99999, sin provocar que se produzca un error.
