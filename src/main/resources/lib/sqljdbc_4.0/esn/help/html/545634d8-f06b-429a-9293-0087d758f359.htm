Método getParameterClassName (SQLServerParameterMetaData)

Recupera el nombre completo de la clase Java cuyas instancias se deberían pasar al método setObject de la clase SQLServerPreparedStatement.

Sintaxis

public java.lang.String getParameterClassName(int param)

Parámetros

param

Un valor int que indica el índice del parámetro.

Valor devuelto

Un String que contiene el nombre completo de la clase.

Excepciones

SQLServerException

Comentarios

El método getParameterClassName especifica este método getParameterClassName en la interfaz java.sql.ParameterMetaData.
