execute

Ejecuta la instrucción SQL en este objeto Statement, que puede ser cualquier tipo de instrucción SQL.

Envía un lote de comandos a la base de datos para que se ejecute. Si todos los comandos se ejecutan correctamente, devuelve una matriz de recuentos de actualizaciones.

executeQuery

Ejecuta la consulta SQL en este objeto Statement y devuelve el objeto SQLServerResultSet que genera la consulta.

executeUpdate

Ejecuta la instrucción SQL en este objeto Statement, que debe ser una instrucción INSERT, UPDATE, MERGE o DELETE de SQL; o una instrucción SQL que no devuelva nada, como una instrucción DDL.
