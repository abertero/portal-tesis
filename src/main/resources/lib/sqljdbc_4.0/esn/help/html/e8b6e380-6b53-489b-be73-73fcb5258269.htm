setDateTimeOffset(int, java.sql.DateTimeOffset) (SQLServerStatement)

Establece el parámetro designado para el valor DateTimeOffset especificado.

Sintaxis

public void setDateTimeOffset(int parameterIndex, DateTimeOffset dateTime)

Parámetros

parameterIndex

Índice de la columna que se va a establecer.

dateTimeOffset

Un objeto DateTimeOffset.

Excepciones

SQLServerException

Comentarios

El formato de DateTimeOffset es "AAAA-MM-DD HH-MM-SS[.nnnnnnn] [+][-] HH:MM". Utilice la siguiente tabla a modo de referencia.

Tipo de SQL

Insert

datetime

Solo puede insertar: "AAAA-MM-DD hh:mm:ss[.nnn]"

smalldatetime

Solo puede insertar: "AAAA-MM-DD hh:mm:ss"

Time

Solo puede insertar: "hh:mm:ss[.nnnnnnn]"

Fecha

Solo puede insertar: "AAAA-MM-DD"

DateTime2

Solo puede insertar: "AAAA-MM-DD hh:mm:ss[.nnnnnnn]"
