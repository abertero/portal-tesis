Método getBytes (SQLServerCallableStatement)

Recupera el valor del parámetro designado como una matriz de bytes.

Lista de sobrecargas

Nombre

Descripción

getBytes (int)

Recupera el valor del parámetro designado como una matriz del valor de bytes según el índice de parámetro.

getBytes (java.lang.String)

Recupera el valor del parámetro designado como una matriz del valor de bytes según el nombre de parámetro.

Notas

En una versión anterior del Controlador Microsoft JDBC para SQL Server, se podía usar SQLServerCallableStatement.getBytes para convertir valores entre las matrices de bytes y los tipos de datos date, time, datetime2 o datetimeoffset de SQL Server. Ahora, al usar este método con esos tipos de datos, se producirá una excepción que indica que no se admite la conversión.
