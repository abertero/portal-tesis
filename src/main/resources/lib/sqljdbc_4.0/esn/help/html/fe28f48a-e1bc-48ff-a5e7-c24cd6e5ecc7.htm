Usar instrucciones con SQL

Cuando se trabaja con datos de una base de datos de SQL Server con el Controlador Microsoft JDBC para SQL Server e instrucciones SQL insertadas, se pueden usar varias clases. La clase usada depende del tipo de instrucción SQL que desee ejecutar.

Si la instrucción SQL no contiene parámetros IN, use la clase SQLServerStatement, pero si contiene parámetros IN, use la clase SQLServerPreparedStatement.

Si necesita usar instrucciones SQL que contengan parámetros IN y OUT, tendrá que implementarlas como un procedimiento almacenado y llamarlo con la clase SQLServerCallableStatement. Para obtener más información sobre el uso de procedimientos almacenados, consulte Usar instrucciones con procedimientos almacenados.
