Conectar y recuperar datos

Cuando se trabaja con el Controlador Microsoft JDBC para SQL Server, hay dos métodos principales para establecer una conexión con una base de datos de SQL Server. Uno de estos métodos establece las propiedades de conexión en la URL de conexión y, a continuación, llama al método getConnection de la clase DriverManager para devolver un objeto SQLServerConnection.

Para obtener una lista de las propiedades de conexión compatibles con el controlador JDBC, consulte Establecer las propiedades de conexión.

El segundo método implica el establecimiento de las propiedades de conexión mediante los métodos establecedores de la clase SQLServerDataSource y, a continuación, la llamada al método getConnection para devolver un objeto SQLServerConnection.

En los temas de esta sección se describen los distintos modos en que puede conectarse a una base de datos de SQL Server y se muestran las distintas técnicas para recuperar datos.
