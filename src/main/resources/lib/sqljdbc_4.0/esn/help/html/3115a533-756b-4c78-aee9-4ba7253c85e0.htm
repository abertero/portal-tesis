Métodos

Nombre

Descripción

clearWarnings

Borra todas las advertencias que se han notificado para este objeto SQLServerConnection.

close

Libera inmediatamente la base de datos para este objeto SQLServerConnection y los recursos de JDBC en lugar de esperar a que se liberen automáticamente.

commit

Realiza todos los cambios efectuados desde la confirmación anterior o realiza una reversión permanente y libera los bloqueos de la base de datos que incorpore este objeto SQLServerConnection en esos momentos.

createBlob

Crea un objeto java.sql.Blob sin datos.

createClob

Crea un objeto java.sql.Clob sin datos.

createNClob

Crea un objeto java.sql.NClob sin datos.

createStatement

Crea un objeto SQLServerStatement para enviar instrucciones SQL a la base de datos.

createSQLXML

Crea un objeto java.sql.SQLXML sin datos.

getAutoCommit

Recupera el modo de confirmación automática actual para este objeto SQLServerConnection.

getCatalog

Recupera el nombre del catálogo actual para este objeto SQLServerConnection.

Método getClientConnectionID (SQLServerConnection)

Obtiene el identificador de conexión del intento de conexión más reciente, independientemente de que dicho intento fuera correcto o erróneo.

getClientInfo

Recupera información con respecto a las propiedades de información del cliente que admita el controlador JDBC.

getHoldability

Recupera la capacidad de alojamiento actual de los objetos SQLServerResultSet que se crean mediante el uso del objeto SQLServerConnection.

getMetaData

Recupera un objeto SQLServerDatabaseMetaData que contiene los metadatos sobre la base de datos para la que este objeto SQLServerConnection representa una conexión.

getTransactionIsolation

Recupera el nivel de aislamiento de transacción actual para este objeto SQLServerConnection.

getTypeMap

Recupera el objeto Map que está asociado a este objeto SQLServerConnection.

getWarnings

Recupera la primera advertencia que notifican las llamadas en este objeto SQLServerConnection.

isClosed

Indica si se ha cerrado este objeto SQLServerConnection.

isReadOnly

Indica si este objeto SQLServerConnection está en modo solo lectura.

isValid

Indica si este objeto SQLServerConnection se ha cerrado y si sigue siendo válido.

nativeSQL

Convierte la instrucción SQL establecida en gramática SQL nativa del servidor de la base de datos.

prepareCall

Crea un objeto SQLServerCallableStatement para llamar a procedimientos almacenados de la base de datos.

prepareStatement

Crea un objeto SQLServerPreparedStatement para enviar las instrucciones SQL parametrizadas a la base de datos.

releaseSavepoint

Quita el objeto SQLServerSavepoint especificado de la transacción actual.

rollback

Deshace todos los cambios realizados en la transacción actual y libera cualquier bloque en la base de datos que contenga en esos momentos este objeto SQLServerConnection.

setAutoCommit

Establece el modo de confirmación automática para este objeto SQLServerConnection en el estado determinado.

setCatalog

Establece el nombre del catálogo especificado para seleccionar un subespacio de la base de datos de este objeto SQLServerConnection donde pueda funcionar.

setClientInfo

Establece el valor de las propiedades de la información del cliente.

setHoldability

Cambia la capacidad de alojamiento de los objetos SQLServerResultSet que se han creado mediante el uso de este objeto SQLServerSavepoint según la capacidad de alojamiento determinada.

setReadOnly

Establece este objeto SQLServerConnection en modo de solo lectura como sugerencia para que el controlador JDBC pueda habilitar las optimizaciones de la base de datos.

setSavepoint

Crea un punto de retorno sin nombre en la transacción actual y devuelve el nuevo objeto SQLServerSavepoint que lo representa.

setTransactionIsolation

Intenta cambiar el nivel de aislamiento de transacción para este objeto SQLServerConnection según el que se haya determinado.

setTypeMap

Instala el objeto TypeMap determinado como la asignación de tipos para este objeto SQLServerConnection.
