Método getBinaryStream (ISQLServerResultSet)

Recupera el valor de la columna designada en la fila actual de este objeto SQLServerResultSet como un flujo binario de bytes sin interpretar.

Lista de sobrecargas

Nombre

Descripción

getBinaryStream (int)

Recupera el valor del índice de la columna designada en la fila actual de este objeto SQLServerResultSet como un flujo binario de bytes sin interpretar.

getBinaryStream (java.lang.String)

Recupera el valor del nombre de la columna designada en la fila actual de este objeto SQLServerResultSet como un flujo binario de bytes sin interpretar.
