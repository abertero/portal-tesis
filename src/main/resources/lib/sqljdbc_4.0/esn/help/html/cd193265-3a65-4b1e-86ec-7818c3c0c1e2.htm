Método setBytes (SQLServerBlob)

Escribe la matriz de bytes determinada en el BLOB comenzando en la posición determinada y posteriormente devuelve el número de bytes escritos.

Lista de sobrecargas

Nombre

Descripción

setBytes (long, byte[])

Escribe la matriz de bytes determinada en el BLOB comenzando en la posición determinada y posteriormente devuelve el número de bytes escritos.

setBytes (long, byte[], int, int)

Escribe todo o parte de la matriz de bytes determinada en el BLOB comenzando en la posición, desplazamiento y longitud determinados; y posteriormente devuelve el número de bytes escritos.
