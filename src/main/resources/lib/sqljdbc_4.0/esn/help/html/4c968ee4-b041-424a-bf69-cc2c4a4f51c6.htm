Método setIntegratedSecurity (SQLServerDataSource)

Establece un valor Boolean que indica si la propiedad integratedSecurity está habilitada.

Sintaxis

public void setIntegratedSecurity(boolean enable)

Parámetros

enable

true si integratedSecurity está habilitada. En caso contrario, es false.

Comentarios

Se establece en "true" para indicar que SQL Server va a usar credenciales de Windows para autenticar al usuario de la aplicación. Si el valor es "true", el Controlador Microsoft JDBC para SQL Server buscará en la memoria caché de credenciales del equipo local suministradas en el equipo o en el inicio de sesión de red. Si el valor es "false", se deben suministrar el nombre de usuario y la contraseña.

Esta propiedad solo se admite en los sistemas operativos de Microsoft Windows.

Para obtener más información sobre el uso de la autenticación integrada, vea Generar URL de conexión.
