Referencia de API del controlador JDBC

El Controlador Microsoft JDBC para SQL Server proporciona una API que se puede usar dentro de código de programación Java para conectar con una base de datos de Microsoft SQL Server e interactuar con ella.

Para obtener información conceptual sobre cómo utilizar el controlador JDBC, vea Introducción al controlador JDBC.

Interfaces

Nombre de la interfaz

Descripción

Interfaz ISQLServerCallableStatement

Le permite especificar el nombre del procedimiento almacenado que se va a llamar junto con los parámetros de entrada y salida.

Interfaz de ISQLServerConnection

Representa una conexión JDBC a una base de datos de SQL Server.

Clase SQLServerDataSource

Representa una lista de propiedades concreta para efectuar la conexión a la base de datos de SQL Server utilizando un objeto ISQLServerConnection.

ISQLServerPreparedStatement

Representa la implementación básica de la funcionalidad de la instrucción preparada de JDBC.

ISQLServerResultSet

Representa un conjunto de resultados JDBC.

ISQLServerStatement

Representa la implementación básica de la funcionalidad de la instrucción de JDBC.

Representa un servicio de generación de objetos para materializar los orígenes de datos de la interfaz Java Naming and Directory Interface (JNDI).

Representa una ejecución de una instrucción SQL que se ha realizado de forma incorrecta o incompleta.

SQLServerResource

SQLServerResultSet
