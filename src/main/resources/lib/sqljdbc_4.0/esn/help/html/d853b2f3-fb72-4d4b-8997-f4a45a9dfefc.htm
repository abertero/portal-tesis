Método setURL (ISQLServerPreparedStatement)

Establece el parámetro designado para el valor URL determinado.

Sintaxis

public final void setURL(int parameterIndex,
                         java.net.URL x)

Parámetros

parameterindex

Un int que indica el número del parámetro.

x

Un objeto URL.

Excepciones

SQLServerException

Comentarios

El método setURL especifica este método setURL en la interfaz java.sql.PreparedStatement.
