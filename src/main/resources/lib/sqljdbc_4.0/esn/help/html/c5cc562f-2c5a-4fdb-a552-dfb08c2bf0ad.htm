Método getObject (SQLServerCallableStatement)

Recupera el valor del parámetro designado como un objeto en el lenguaje de programación Java.

Lista de sobrecargas

Nombre

Descripción

getObject (int)

Recupera el valor del parámetro designado como un objeto en el lenguaje de programación Java según el índice del parámetro.

getObject (int, java.util.Map)

Recupera el valor del parámetro designado como un objeto en el lenguaje de programación Java según el índice de parámetro y mediante el uso del objeto Map determinado.

getObject (java.lang.String)

Recupera el valor del parámetro designado como un objeto en el lenguaje de programación Java según el nombre del parámetro.

getObject (java.lang.String, java.util.Map)

Recupera el valor del parámetro designado como un objeto en el lenguaje de programación Java según el nombre de parámetro y mediante el uso del objeto Map determinado.
