updateDateTimeOffset (SQLServerResultSet)

Este método se agregó en el controlador JDBC 3.0 de Microsoft SQL Server.

Actualiza el valor de la columna especificada a un valor Clase DateTimeOffset.

Lista de sobrecargas

Nombre

Descripción

updateDateTimeOffset(int, microsoft.sql.DateTimeOffset)

Actualiza el valor de la columna especificada para el valor Clase DateTimeOffset, según un ordinal de columna basado en cero.

updateDateTimeOffset(string, microsoft.sql.DateTimeOffset)

Actualiza el valor de la columna especificado para el valor Clase DateTimeOffset, según un nombre de columna.
