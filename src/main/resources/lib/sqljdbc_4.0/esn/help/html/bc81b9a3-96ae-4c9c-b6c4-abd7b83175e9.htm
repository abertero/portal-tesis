Método getBigDecimal (SQLServerCallableStatement)

Recupera el valor del parámetro designado como java.math.BigDecimal.

Lista de sobrecargas

Nombre

Descripción

getBigDecimal (int)

Recupera el valor del parámetro designado como java.math.BigDecimal con precisión completa según el índice de parámetro.

getBigDecimal (int, int)

(En desuso) recupera el valor del parámetro designado como java.math.BigDecimal según el índice de parámetro y la escala.

getBigDecimal (java.lang.String)

Recupera el valor del parámetro designado como java.math.BigDecimal con precisión completa según el nombre de parámetro.

getBigDecimal (java.lang.String, int)

(En desuso) recupera el valor del parámetro designado como java.math.BigDecimal según el nombre de parámetro y la escala.
