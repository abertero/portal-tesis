Método isWrapperFor (SQLServerConnectionPoolDataSource)

Indica si este objeto es un contenedor para la interfaz especificada.

Sintaxis

public boolean isWrapperFor(Class iface)

Parámetros

iface

Un valor class que define una interfaz.

Valor devuelto

true si este objeto implementa la interfaz o contiene un objeto que implemente la interfaz. En caso contrario, es false.

Excepciones

SQLServerException

Comentarios

El método isWrapperFor y el métodounwrap se definen a través de la interfaz java.sql.Wrapper, que se incorpora en las especificaciones de JDBC 4.0.

Si este método devuelve el valor TRUE, la llamada a unwrap con el mismo argumento se realizará correctamente.

Para obtener más información, vea Contenedores e interfaces.
