Método setClientInfo (java.util.Properties)

Establece el valor de las propiedades de información de cliente de la conexión.

Sintaxis

public void setClientInfo (java.util.Properties properties)

Parámetros

properties

Un objeto Properties que contiene la lista de propiedades de información de cliente que se van a establecer.

Excepciones

SQLServerException

Comentarios

El método setClientInfo especifica este método setClientInfo en la interfaz java.sql.Connection.

El Controlador Microsoft JDBC para SQL Server no admite propiedades de información de cliente. Este método genera advertencias si el parámetro de entrada properties no hace referencia a un conjunto de propiedades vacías. Es decir, este método genera advertencias para las propiedades que desea establecer la aplicación. Las aplicaciones deberían utilizar el método getWarnings de la clase SQLServerConnection para recuperar cada advertencia.
