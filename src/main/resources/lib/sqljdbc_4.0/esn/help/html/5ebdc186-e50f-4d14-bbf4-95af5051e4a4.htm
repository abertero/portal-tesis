addBatch

(Se hereda de SQLServerPreparedStatement). Agrega un conjunto de parámetros al lote de comandos para este objeto CallableStatement.

cancel

(Se hereda de SQLServerStatement). Cancela la instrucción de SQL que se esté ejecutando en esos momentos este objeto CallableStatement.

clearBatch

(Se hereda de SQLServerPreparedStatement). Vacía la lista actual de comandos SQL para este objeto CallableStatement.

clearParameters

(Se hereda de SQLServerPreparedStatement). Borra inmediatamente los valores de parámetro actuales.

clearWarnings

(Se hereda de SQLServerStatement). Borra todas las advertencias que se notifican en este objeto CallableStatement.

close

(Se hereda de SQLServerPreparedStatement). Libera inmediatamente los recursos de la base de datos y de JDBC de este objeto CallableStatement en lugar de esperar a que se liberen automáticamente.

execute

(Se hereda de SQLServerPreparedStatement). Ejecuta la instrucción SQL en este objeto CallableStatement, que puede ser cualquier tipo de instrucción SQL.

executeBatch

(Se hereda de SQLServerPreparedStatement). Envía un lote de comandos a la base de datos para que se ejecute. Si todos los comandos se ejecutan correctamente, devuelve una matriz de recuentos de actualizaciones.

executeQuery

(Se hereda de SQLServerPreparedStatement). Ejecuta la consulta SQL en este objeto CallableStatement y devuelve el objeto SQLServerResultSet que genera la consulta.

executeUpdate

(Se hereda de SQLServerPreparedStatement). Ejecuta la instrucción SQL en este objeto CallableStatement, que debe ser una instrucción INSERT, UPDATE, MERGE o DELETE de SQL; o una instrucción SQL que no devuelva nada, como una instrucción DDL.
