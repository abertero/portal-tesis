Usar metadatos de una base de datos

Para consultar en una base de datos información sobre compatibilidad, el Controlador Microsoft JDBC para SQL Server implementa la clase SQLServerDatabaseMetaData. Esta clase contiene numerosos métodos que devuelven información en forma de un solo valor o de conjunto de resultados.

Para crear un objeto SQLServerDatabaseMetaData, puede utilizar el método getMetaData de la clase SQLServerConnection para obtener información sobre la base de datos a la que está conectado.

En el ejemplo siguiente, una conexión abierta a la base de datos de ejemplo SQL Server 2005AdventureWorks se pasa a la función, el método getMetaData de la clase SQLServerConnection se utiliza para devolver un objeto SQLServerDatabaseMetadata y, a continuación, se usan los diversos métodos del objeto SQLServerDatabaseMetaData para mostrar información sobre el controlador, su versión y el nombre y la versión de la base de datos.

public static void getDatabaseMetaData(Connection con) {
   try {
      DatabaseMetaData dbmd = con.getMetaData();
      System.out.println("dbmd:driver version = " + dbmd.getDriverVersion());
      System.out.println("dbmd:driver name = " + dbmd.getDriverName());
      System.out.println("db name = " + dbmd.getDatabaseProductName());
      System.out.println("db ver = " + dbmd.getDatabaseProductVersion());
   }
   catch (Exception e) {
      e.printStackTrace();
   }
}
