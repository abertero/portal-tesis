Método setNClob (java.lang.String, java.sql.NClob)

Establece el parámetro designado en el objeto NClob especificado.

Sintaxis

public final void setNClob(java.lang.String parameterName,
              java.sql.NClob value)

Parámetros

parameterName

Un String que contiene el nombre del parámetro.

value

Un objeto NClob.

Excepciones

SQLServerException

Comentarios

Este método se debería utilizar para NCHAR, NVARCHAR, NTEXT y los tipos de datos de parámetro XML.

El método setNClob especifica este método setNClob en la interfaz java.sql.CallableStatement.
