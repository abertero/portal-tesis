Método first (ISQLServerResultSet)

Mueve el cursor a la primera fila de este objeto SQLServerResultSet.

Sintaxis

public boolean first()

Valor devuelto

Es true si el cursor se desplaza a la primera fila. En caso contrario, es false.

Excepciones

SQLServerException

Comentarios

El método first especifica este método first en la interfaz java.sql.ResultSet.
