Método getCatalogSeparator (SQLServerDatabaseMetaData)

Recupera el valor String que esta base de datos emplea como separador entre un nombre de catálogo y de tabla.

Sintaxis

public java.lang.String getCatalogSeparator()

Valor devuelto

Un valor String que contiene el separador del catálogo.

Excepciones

SQLServerException

Comentarios

El método getCatalogSeparator especifica este método getCatalogSeparator en la interfaz java.sql.DatabaseMetaData.

Cuando se usa el Controlador Microsoft JDBC para SQL Server con una base de datos de SQL Server, este método devuelve un punto (".") como separador del catálogo.
