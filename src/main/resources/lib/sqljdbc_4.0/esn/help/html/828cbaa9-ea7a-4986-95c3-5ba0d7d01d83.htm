Métodos

Nombre

Descripción

addBatch

Agrega el comando SQL determinado a la lista de comandos actual para este objeto SQLServerStatement.

cancel

Cancela la instrucción de SQL que se esté ejecutando en esos momentos en este objeto SQLServerStatement.

clearBatch

Vacía la lista actual de comandos SQL para este objeto SQLServerStatement.

clearWarnings

Borra todas las advertencias que se notifican en este objeto SQLServerStatement.

close

Libera inmediatamente la base de datos de este objeto SQLServerStatement y los recursos de JDBC en lugar de esperar a que se liberen automáticamente.

execute

Ejecuta la instrucción SQL determinada, que puede devolver varios resultados.

executeBatch

Envía un lote de comandos a la base de datos para que se ejecute. Si todos los comandos se ejecutan correctamente, devuelve una matriz de recuentos de actualizaciones.

executeQuery

Ejecuta la instrucción SQL determinada y devuelve un objeto SQLServerResultSet único.

executeUpdate

Ejecuta la instrucción SQL determinada, que puede ser una instrucción INSERT, UPDATE, MERGE o DELETE; o una instrucción SQL que no devuelve nada, como una instrucción DDL de SQL.

getConnection

Recupera el objeto SQLServerConnection que generó este objeto SQLServerStatement.

getFetchDirection

Recupera la dirección para capturar las filas de las tablas de base de datos que es el valor predeterminado para los conjuntos de resultados que se generaron a partir de este objeto SQLServerStatement.

getFetchSize

Recupera el número de filas del conjunto de resultados que sea el tamaño de captura predeterminado para los objetos del conjunto de resultados que generó este objeto SQLServerStatement.

getGeneratedKeys

Recupera cualquiera clave generada automáticamente que se cree como resultado de ejecutar este objeto SQLServerStatement.

getMaxFieldSize

Recupera el número máximo de bytes que se pueden devolver para valores de columna de caracteres y binarios en un objeto SQLServerResultSet que haya generado este objeto SQLServerStatement.

getMaxRows

Recupera el número máximo de filas que puede contener un objeto SQLServerResultSet que haya generado este objeto SQLServerStatement.

getMoreResults

Se desplaza al próximo resultado de este objeto SQLServerStatement.

getQueryTimeout

Recupera el número de segundos que el Controlador Microsoft JDBC para SQL Server esperará la ejecución de este objeto SQLServerStatement.

getResponseBuffering

Recupera el modo de almacenamiento en búfer de respuesta para este objeto SQLServerStatement.

getResultSet

Recupera el resultado actual como un objeto SQLServerResultSet.

getResultSetConcurrency

Recupera la simultaneidad del conjunto de resultados para los objetos SQLServerResultSet que haya generado este objeto SQLServerStatement.

getResultSetHoldability

Recupera la capacidad de alojamiento del conjunto de resultados para los objetos SQLServerResultSet que haya generado este objeto SQLServerStatement.

getResultSetType

Recupera el tipo del conjunto de resultados para los objetos SQLServerResultSet que haya generado este objeto SQLServerStatement.

getUpdateCount

Recupera el resultado actual como un recuento de actualizaciones.

getWarnings

Recupera la primera advertencia que notifican las llamadas en este objeto SQLServerStatement.

isClosed

Indica si se ha cerrado este objeto SQLServerStatement.

isPoolable

Devuelve un valor que indica si una instrucción se puede agregar al grupo de instrucciones proporcionado por el usuario.

isWrapperFor

Indica si este objeto de instrucción es un contenedor para la interfaz especificada.

setCursorName

Establece el nombre de cursor de SQL para la cadena determinada, el cual utilizarán métodos ulteriores.

setEscapeProcessing

Establece el modo de procesamiento de escape.

setFetchDirection

Ofrece una sugerencia al controlador JDBC sobre la dirección en la que se deberían procesar las filas del conjunto de resultados.

setFetchSize

Ofrece una sugerencia al controlador JDBC sobre el número de filas que se deberían capturar en la base de datos cuando se necesitan más filas.

setMaxFieldSize

Establece el límite para el número máximo de bytes en una columna SQLServerResultSet que almacena valores de caracteres o binarios en el número establecido de bytes.

setMaxRows

Establece el límite para el número máximo de filas que cualquier objeto SQLServerResultSet puede contener para el número determinado.

setPoolable

Solicita que una instrucción se agrupe o no.

setQueryTimeout

Establece el número de segundos que esperará el controlador antes de que se ejecute un objeto SQLServerStatement según el número de segundos determinado.

setResponseBuffering

Establece el modo de almacenamiento en búfer de respuesta para este objeto SQLServerStatement en String full con distinción entre mayúsculas y minúsculas o en adaptive.

unwrap

Devuelve un objeto que implementa la interfaz especificada para permitir el acceso a los métodos específicos del Controlador Microsoft JDBC para SQL Server.
