Método getClob (int) (ISQLServerResultSet)

Recupera el valor del índice de la columna designada en la fila actual de este objeto SQLServerResultSet como un objeto Clob en el lenguaje de programación Java.

Sintaxis

public java.sql.Clob getClob(int columnIndex)

Parámetros

columnIndex

Un int que indica el índice de la columna.

Valor devuelto

Un objeto Clob.

Excepciones

SQLServerException

Comentarios

El método getClob especifica este método getClob en la interfaz java.sql.ResultSet.
